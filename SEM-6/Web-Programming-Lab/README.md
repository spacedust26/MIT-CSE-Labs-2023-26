### Creating a Virtual Environment
- Linux

```
sudo apt-get install python3-venv  
python3 -m venv .venv
source .venv/bin/activate
```

- macOS
```
python3 -m venv .venv
source .venv/bin/activate
```

- Windows
```
py -3 -m venv .venv
.venv\scripts\activate
```
